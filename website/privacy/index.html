Privacy

Last updated: 2026-06-20

Overview

This public website does not require an account and does not use advertising pixels. Optional analytics are disabled unless a visitor explicitly allows them.

Optional Google Analytics

Google Analytics 4 uses measurement ID G-5NBSSFH77M only after a visitor selects Allow analytics. Before consent, and after selecting Decline, the analytics tag is not added to the page.

When allowed, analytics may process page, referrer, browser, device, approximate region, and interaction information. Google signals and advertising-personalization signals are disabled.

Analytics choice

The first-party cookie doll_analytics stores granted or denied so the choice can be remembered. The footer control can be used to change it.

Hosting and external services

Cloudflare Pages hosts and protects the site. The homepage also requests public development data from the GitHub public API. The contact link opens a Google Form. These services process normal request information under their own privacy terms.

A short-lived browser cache named doll-development-log-v1 may be used to reduce repeated GitHub API requests.

Contact

Privacy questions can be sent through the contact form.
